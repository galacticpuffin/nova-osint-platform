# NOVA OSINT Platform

Built by galacticpuffin

NOVA is a work-in-progress full-stack OSINT platform focused on digital footprint analysis, exposure awareness, and privacy-focused workflows.

I’m building this to better understand how publicly available data can be collected, connected, and interpreted in a way that’s actually useful to real people.

## Overview

This project is my way of combining security research, system design, and practical engineering into something real.

The focus is on how identity signals, public data, and fragmented information across the internet can be pulled together and analyzed in one place. A lot of the thinking behind this is influenced by OSINT workflows and how analysts approach investigations.

## What I’ve worked on so far

I’ve been building this end-to-end, including the frontend, backend logic, and data flow.

Current focus areas:
- API-driven data collection and enrichment
- structuring and normalizing data across different sources
- designing how results are surfaced and interpreted
- early risk scoring and exposure concepts
- experimenting with AI and agentic workflows to automate parts of the process

## Tech

React and Next.js on the frontend  
Serverless functions for backend logic  
Vercel for deployment  
Supabase for data and workflow support  
External APIs for enrichment and OSINT-style lookups  
Early AI/agentic systems for workflow automation
